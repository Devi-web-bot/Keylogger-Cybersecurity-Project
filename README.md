# Keylogger Project (Cybersecurity)

## Description
This project demonstrates a Python-based keylogger using Tkinter and pynput.
It captures keystrokes and stores them in text and JSON files.

⚠️ This project is strictly for educational and cybersecurity awareness purposes.

## Technologies Used
- Python
- Tkinter
- pynput
- JSON

## Features
- GUI-based keylogger
- Logs keystrokes
- Saves output in logs.txt and log.json
